# BUILD-A-THON-IOT-Health-Monitoring-System-
Developed an IoT-based Health Monitoring System using ESP32 and multiple biomedical sensors to monitor Heart Rate, SpO₂, ECG, Pulse Rate, and Glucose levels in real time. The system displays data on an OLED screen and transmits it to Blynk and Google Sheets for remote monitoring, cloud storage, and future health analysis. 🚀📊❤️
